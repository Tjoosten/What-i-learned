HTTP Codes
======================

## 1xx Informational

| Code: | Description:          |
| :---- | :-------------------- |
| `100` | Continue              |
| `101` | Switching protocols   |
| `102` | Processing (RFC 2518) |

## 2xx Success

| Code: | Description:                  |
| :---- | :---------------------------- |
| `200` | OK                            |
| `201` | Created                       |
| `202` | Accepted                      |
| `203` | Non-Authoritative Information |
